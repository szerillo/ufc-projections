# ufc-projections

GitHub Pages site hosting UFC projection widgets for embedding in WordPress.

**Live site:** https://szerillo.github.io/ufc-projections/

---

## WordPress Embed Codes

### Moneyline Projections

Paste this into a WordPress **Custom HTML** block:

```html
<iframe src="https://szerillo.github.io/ufc-projections/moneylines.html" width="100%" height="500" frameborder="0" scrolling="auto" style="border:none;"></iframe>
```

### Props Projections

Paste this into a WordPress **Custom HTML** block:

```html
<iframe src="https://szerillo.github.io/ufc-projections/props.html" width="100%" height="600" frameborder="0" scrolling="auto" style="border:none;"></iframe>
```
